# Playful Sparkle - Quantity Increment Decrement for OpenCart 3

The **Playful Sparkle - Quantity Increment Decrement** extension for OpenCart 3.x+ enhances the shopping experience by adding intuitive plus and minus buttons to product quantity inputs. This allows customers to easily adjust the number of items they wish to purchase without manual typing, streamlining the path to checkout. With extensive customization options, store owners can perfectly match the appearance of these buttons to their store's branding, choosing from various icon styles, color themes, and custom CSS overrides.

---

## Features

- **Visual Customization**: Choose from multiple icon sets including plus / minus (+ / -), chevron left / right (‹ ›), arrow up / down (↑ ↓).
- **Flexible Styling**: Select from standard Bootstrap color themes (Primary, Success, Danger, etc.) or enable manual overrides for complete control over colors.
- **Comprehensive Placement**: Enable the increment/decrement functionality across various sections of your store, including category page, product detail page, compare products page, shopping cart page, wishlist page, manufacturer / brand page, search results page, specials page, specials module, latest products module, bestsellers module and the featured products module.
- **Button State Management**: Independently customize colors for different increment/decrement button states: default, hover, active, and disabled.
- **Quantity Input Integration**: Enhances text inputs with a user-friendly quantity adjustment interface.
- **Native Layout Integration**: Nicely incorporates into your webshop by utilizing the standard OpenCart layout structures and Bootstrap grid system for a seamless look and feel.
- **Compatibility**: Integrates smoothly with OpenCart 3.x+, supporting a range of online store setups.
- **Multilingual Support**: Ready for international use with languages including العربية (ar), فارسی (fa-ir), Български (bg), 中文(简体) (zh-cn), 中文(繁體) (zh-tw), Čeština (cs-cz), English (UK) (en-gb), English (US) (en-us), Français (fr-fr), Deutsch (de-de), Ελληνικά (el-gr), Magyar (hu-hu), Italiano (it-it), 日本語 (ja), 한국어 (ko-kr), Polski (pl-pl), Português (Brasil) (pt-br), Русский (ru-ru), Slovenčina (sk-sk), Español (es-es), and Türkçe (tr-tr).

## Extension Visibility

You can choose to display the quantity buttons on a per-page or per-module basis:

- **Pages**: Category, product detail, compare products, shopping cart, wishlist, manufacturer / brand, search results and specials pages.
- **Modules**: Specials, latest products, bestsellers and featured products modules.

## Installation Instructions

1. Download the latest version from this repository.
2. Log in to your OpenCart admin panel.
3. Navigate to `Extensions > Installer`.
4. Click the `Upload` button and upload the `ps_quantity_increment_decrement.ocmod.zip` file.
5. Navigate to `Extensions > Extensions` and select `Modules` from the `Choose the extension type` dropdown list.
6. Locate the extension in the `Modules` list and click the `Install` button.
7. Click the `Edit` button, configure the extension parameters, and click the `Save` button to save your settings.
8. Navigate to `Extensions > Modifications`, select all your modifications and click the `Refresh` button to update the modification cache and apply the changes.
